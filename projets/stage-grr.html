GRR — Gestionnaire de Réservation de Ressources

Déploiement et configuration d'un système GRR open-source sur WAMP pour la gestion des réservations de ressources informatiques sur le réseau interne.

PHP MySQL WAMP Réseau Windows

Contexte

La DSDEN du Morbihan avait besoin d'un système centralisé pour que les équipes puissent réserver du matériel informatique (salles, équipements) sans passer par des échanges informels. J'ai déployé et configuré le logiciel open-source GRR (Gestion et Réservation de Ressources).

Ce que j'ai fait

  • Installation et configuration de WAMP Server sur Windows 11 (serveur local)
  • Installation des prérequis Visual C++ Redistributable
  • Déploiement de GRR depuis GitHub (JeromeDevome/GRR) dans le répertoire /reservations
  • Création de la base de données MySQL grr via phpMyAdmin
  • Configuration Apache pour accès réseau : modification de httpd.conf et des VirtualHosts
  • Ouverture du pare-feu Windows sur le port 80 (TCP)
  • Configuration du fichier hosts pour nommage interne : infra56/reservations
  • Rédaction d'un guide d'installation complet pour la reprise et maintenance

Dépannage résolu

  • Port 80 occupé (Skype, IIS) → identification et libération du port
  • Erreur 403 Apache → correction des droits d'accès (Require all granted)
  • Accès réseau impossible → configuration VirtualHost + règles pare-feu

Résultat

Le GRR est accessible depuis tous les postes du réseau interne via http://infra56/reservations. Un guide d'installation complet a été remis pour faciliter la reprise du service.
